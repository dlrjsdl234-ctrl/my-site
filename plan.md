# 계산기 카테고리에 X 계산기 부분에 기본값셋팅
## 환포계산기
원하는 환포 "원하는 환포를 입력하세요 예)30000000" 플레이스홀더 적용
RP상점 환포 400은 기본값으로사용하자
도감수치 120은 기본값으로사용하자
유물 반지 "유물반지 퍼센트를 입력하세요 예)20" 플레이스홀더 적용
VIP 10은 기본값으로
초월 레벨 "초월 레벨을 입력하세요 예)0" 플레이스홀더 적용
모래시계레벨은 기본값으로 0
내 현재 레벨 "현재 레벨을 입력하세요/ 레벨로 RP계산시 필요 / 예)200 " 플레이스홀더 적용

##경험치 계산기

목표레벨 "목표 레벨을 입력하세요 예) 455" 플레이스홀더 적용

모래시계 레벨 기본값으로 0
